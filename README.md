Number Guessing Game:
A fun Python game where you try to guess a random number between 0 and 10! The game tells you if your guess is too high or too low. Keep guessing until you find the correct number!
How to Play
Run the program:
Enter a number between 0 and 10
Get hints: "Too High" or "Too Low"
Keep guessing until you get it right!
See how many steps it took
Play again or quit
Example:
Enter your Number : 7
Your Guess Is too High.
Enter your Number : 4
Your Guess is too Low.
Enter your Number : 5
Number of step taken to reach to the Number : 2
The Numbers you print are [7, 4]
You guessed the Number and the Number Was : 5
Do you want another Attempt : n
Features:
Random number generation
Helpful hints (too high/too low)
Tracks all your guesses
Shows number of attempts
Play multiple rounds
